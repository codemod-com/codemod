Workflows on the Codemod Platform let you automate, coordinate, and track complex code modifications across your organization.

<Tip>
Workflows are available in beta for Codemod [enterprise](https://codemod.com/pricing) customers.
</Tip>

## What are Workflows?

Workflows are sequences of codemods, scripts, or manual steps that can be run across multiple repositories or projects. With workflows, you can define define multi-step processes that, automate code changes, ensure consistency & compliance, and more.

## Running a Workflow

<Frame>
    <CodemodWorkflowsDemo />
</Frame>

<Steps>
  <Step title="Go to Workflows">
    Navigate to the [Workflows section](https://app.codemod.com/workflows/create) in the Codemod web app.
  </Step>
  <Step title="Create a Workflow">
    Click "Create Workflow" and provide a Workflow and project name.
  </Step>
  <Step title="Select repository">
    Select the repository you want the workflow to run over.
  </Step>
  <Step title="Create & Run workflow">
    Click Create Workflow -> Run Automation (or Try Automation for Free) to create the workflow.
  </Step>
</Steps>
